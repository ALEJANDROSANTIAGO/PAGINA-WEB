Bienvenidos, en este espacio Usted encontrará información acerca de la estructura general del Curso Diseño de Sitios Web, un curso diseñado para el desarrollo de la investigación y el aprendizaje autónomo. Los sitios web son herramientas cuya finalidad es: informar, vender o anunciar productos o servicios, recrear, intercambiar ideas sobre un tema de interés general o un gusto en particular; estos se componen de varias páginas web desarrolladas en lenguaje html, que se ingresan en un mismo dominio, para ser publicados en la Word Wide Web, y de esta manera los usuarios pueden acceder a su contenido. Un sitio web se compone de una página índex o página de inicio en la cual se alojan los diversos contenidos. Una página web es un documento de tipo electrónico en el cual se puede alojar información de tipo visual y sonora, por medio de textos audios y videos. CONTENIDO

• DEFINICION Y CONCEPTOS BASICO DE GIT Y GITHUB

GIT y GITHUB son totalmente diferentes.

QUE ES GIT: Es un software de control de versiones diseñado por Linus Torvalds, pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente. Es el software que rastrea. Le da seguimiento a todos los cambios que se ejecutan sobre un archivo o carpeta. Cada cambio que se realice en un directorio, GIT se da cuenta y lo registra.

CARACTERISTICAS DE GIT.

1. Es un sistema de control de versiones distribuido. Con esto, nos referimos a que GIT clona los proyectos para que cada persona ó miembro de un equipo tenga una copia exacta y completa de todo el código, historial y las personas que estuvieron involucradas. El registro encapsulado de todos los cambios de estos elementos se le conoce como repositorio, mencionado anteriormente. Si se llegase a perder el repositorio original, no habría mucho drama porque es probable que existan personas que tienen un clón. De ahí, se puede partir sin problema.

2. Es Open Source. GIT es gratuito, puedes instalarlo en cualquier ordenador o servidor 3. Puedes utilizarlo offline. Si vas en un avión, puedes seguir trabajando en tu proyecto, en local, para posteriormente cuando te conectes a Internet, puedas subirlo al repositorio principal. QUE ES GITHU: Es la plataforma del hosting de los proyectos. Una comunidad llena de personas que desarrollan y comparten usando GIT. Desde enero de 2010, GitHub opera bajo el nombre de GitHub, Inc. El código se almacena de forma pública, aunque también se puede hacer de forma privada, creando una cuenta de pago.
